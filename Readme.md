# Sistema de Gestão de Estoque

Este é um projeto pessoal desenvolvido com o objetivo de praticar e demonstrar conhecimentos em programação com C# e .NET. O sistema realiza o controle simples de estoque, permitindo:

- Cadastro de produtos;
- Consulta de produtos por nome ou código;
- Registro de entrada e saída de estoque;
- Exibição de produtos com estoque baixo.
